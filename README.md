# Roha

Roha is a personal AI assistant built from scratch in Python.

## Vision

Roha is designed to become an intelligent personal assistant capable of:

- Natural conversation
- Long-term memory
- Voice interaction
- Computer automation
- Web research
- Coding assistance
- Document understanding
- Personal productivity

## Tech Stack

- Python
- Ollama
- Local LLMs
- SQLite (initially)
- ChromaDB (later)
- Whisper
- Piper
